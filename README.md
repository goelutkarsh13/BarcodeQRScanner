# BarcodeQRScanner
A scanning tool that reads barcodes and QR codes using the device camera.

## Features
- Barcode + QR recognition  
- Instant decode output  
- Supports multiple formats  

## How to Run
1. Import `.aia` file into MIT App Inventor.
2. Connect device with camera.
3. Run via **AI Companion**.

## Tech Stack
- MIT App Inventor  
- BarcodeScanner component
